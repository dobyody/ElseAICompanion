Ce este memoria virtuală?
Memoria virtuală permite proceselor să adreseze mai multă memorie decât cea fizică disponibilă. Se implementează prin paginare și segmentare.

La accesarea unei pagini absente din RAM, se produce un page fault, iar SO încarcă pagina de pe disc.
